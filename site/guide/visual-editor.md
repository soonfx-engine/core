# Visual Editor

The Visual Editor is the core of the SoonFx workflow.

> **Note**: Documentation for the Visual Editor is currently being updated. Please refer to the [Editor Repository](https://github.com/soonfx-engine/editor) for more information.

## Overview

The editor allows you to:

1.  Create **Nodes** representing game entities (e.g., Hero, Skill).
2.  Define **Attributes** (e.g., Attack, Defense).
3.  Connect attributes with **Formulas**.
