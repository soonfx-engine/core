### Prerequisites

- Node.js >= 14.0.0
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local server URL (usually `http://localhost:3009` or similar).
